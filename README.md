## ➰ 프로젝트 페이지 구성 및 기능

1. `메인 페이지`
   <img src="메인.gif" width="600px"/>
   <br />

   - Carousel (배너, 추천 상품)
   - 하루 특가 타임 기능
   - 카테고리 2depth 구현
   - 카테고리 페이지 이동
  
<br />

2. `회원가입` , `로그인`
   <img src="로그인.gif" width="600px"/>
   <br />

   - 중복확인 기능
   - 유효성 검사
   - 소셜 로그인 (구글 로그인)
   - 로그인 시에는 메인페이지 OO님, 로그아웃 추가

<br />

3. `카테고리 페이지`
   <img src="카테고리 페이지.gif" width="600px"/>
   <br />

   - 추천순 / 낮은 가격순 / 높은 가격순 필터링
   - 페이지네이션
     
<br />

4. `디테일 페이지`
   <img src="기타.gif" width="600px"/>
   <br />
   - 검색창 키워드 입력 시 상품명 자동 완성
   - 상품 상세 페이지
   - 위시리스트 클릭 시 모달 및 이동
   - 장바구니 담기 클릭 시 모달 및 이동
